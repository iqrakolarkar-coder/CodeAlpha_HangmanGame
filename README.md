# Hangman Game 🎮

This is a simple text-based Hangman game built using Python.

## Features
- Random word selection
- Letter guessing and full word guessing
- Limited wrong attempts
- Replay option
- Console-based gameplay

## Technologies Used
- Python
- Random module
- Loops and conditional statements

## How to Run
1. Download the file hangman.py
2. Run in terminal:
   python hangman.py
